# Nodewire Scaffolding Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up a buildable, runnable Forge 1.20.1 Kotlin mod skeleton named Nodewire at `~/CODING/nodewire/` that loads alongside the user's Total Valkyrien instance modset and registers a placeholder `logic_block`.

**Architecture:** Standard ForgeGradle 6 single-module project. Kotlin entry via Kotlin for Forge 4.12.0 (`modLoader="kotlinforforge"`). Hard runtime deps on Valkyrien Skies 2 and Create are declared in `mods.toml` and pulled via maven (`fg.deobf`). Dev client uses a symlink from `run/mods` to the PrismLauncher instance's mods directory, so the dev environment sees every jar the user already has.

**Tech Stack:** Minecraft 1.20.1 · Forge 47.4.10 · Gradle 8.8 · ForgeGradle 6.0.+ · Kotlin 2.0.20 · Kotlin for Forge 4.12.0 · Java 17 · Official Mojang mappings.

**Reference spec:** [`../specs/2026-05-12-nodewire-scaffolding-design.md`](../specs/2026-05-12-nodewire-scaffolding-design.md)

**Verification model:** This plan creates project infrastructure, not application logic. There is no unit test framework yet — "verification" steps are `./gradlew` builds and a manual `runClient` smoke check against the 7 acceptance criteria in the spec. Treat each verification step like a passing test.

## Task 6: Registry + placeholder `logic_block`

- [ ] **Step 6.1: Write `Registry.kt`**

```kotlin
package dev.nitka.nodewire

import net.minecraft.core.registries.Registries
import net.minecraft.resources.ResourceLocation
import net.minecraft.world.item.BlockItem
import net.minecraft.world.item.CreativeModeTab
import net.minecraft.world.item.CreativeModeTabs
import net.minecraft.world.item.Item
import net.minecraft.world.level.block.Block
import net.minecraft.world.level.block.Blocks
import net.minecraft.world.level.block.state.BlockBehaviour
import net.minecraftforge.event.BuildCreativeModeTabContentsEvent
import net.minecraftforge.eventbus.api.IEventBus
import net.minecraftforge.registries.DeferredRegister
import net.minecraftforge.registries.ForgeRegistries
import net.minecraftforge.registries.RegistryObject

object Registry {
    private val BLOCKS: DeferredRegister<Block> =
        DeferredRegister.create(ForgeRegistries.BLOCKS, Nodewire.ID)
    private val ITEMS: DeferredRegister<Item> =
        DeferredRegister.create(ForgeRegistries.ITEMS, Nodewire.ID)

    val LOGIC_BLOCK: RegistryObject<Block> = BLOCKS.register("logic_block") {
        Block(BlockBehaviour.Properties.copy(Blocks.IRON_BLOCK))
    }

    val LOGIC_BLOCK_ITEM: RegistryObject<Item> = ITEMS.register("logic_block") {
        BlockItem(LOGIC_BLOCK.get(), Item.Properties())
    }

    fun register(bus: IEventBus) {
        BLOCKS.register(bus)
        ITEMS.register(bus)
        bus.addListener(::onBuildTabs)
    }

    private fun onBuildTabs(event: BuildCreativeModeTabContentsEvent) {
        if (event.tabKey == CreativeModeTabs.REDSTONE_BLOCKS) {
            event.accept(LOGIC_BLOCK_ITEM.get())
        }
    }
}
```
